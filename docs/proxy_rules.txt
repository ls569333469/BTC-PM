# ================================================================
#  BTC-PM 项目 · Clash 规则模式配置清单
#  更新时间: 2026-03-27
# ================================================================
#
#  本清单列出项目运行时所有外部网络请求，按地域限制分级标注。
#  "日本节点" 指使用日本 IP 出口的代理服务器。
#
#  ⚠️  选节点核心原则：
#  - Polymarket: 受 OFAC 制裁限制，屏蔽美国 IP → 必须用非美国节点
#  - Binance:    屏蔽中国大陆 IP，但日本/新加坡/香港均可
#  - 综合最优解：统一走【日本节点】可同时满足所有服务

# ================================================================
# 🛑 DIRECT (直连) — 绝对不能走代理
# ================================================================
# 原因：TUN/系统代理会拦截本地回环通信，导致前端 502 / Failed to load analysis
#
# Clash rules:
#   - DOMAIN,localhost,DIRECT
#   - IP-CIDR,127.0.0.0/8,DIRECT,no-resolve
#   - IP-CIDR6,::1/128,DIRECT,no-resolve
#
# 涉及网址:
#   http://localhost:5173    ← 前端 Vite 开发服务器
#   http://localhost:8000    ← 后端 FastAPI

# ================================================================
# 🇯🇵 PROXY — 必须走日本节点
