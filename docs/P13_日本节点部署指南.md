# P13 · BTC-PM 量化系统：日本节点上云部署指南

针对本系统（Vite 前端 + FastAPI 后端 + NumPy/Chanpy 计算引擎），考虑到币安 API 低延迟优势，**强烈推荐部署在日本东京节点**。

> 📌 Google Cloud 东京方案详见 → [P14_Google_Cloud_Tokyo_部署方案.md](P14_Google_Cloud_Tokyo_部署方案.md)

---

## 💻 1. 硬件配置要求

*   **OS**：**Ubuntu 22.04 LTS**（避免 CentOS/Alpine，TA-Lib 编译兼容性差）
*   **最低配置**：2 核 / 2GB RAM / 40GB SSD（需设 Swap）
*   **推荐配置**：2 核 / 4GB RAM / 60GB SSD

## 🚀 4. 部署步骤（SSH 登录后执行）

### Step 1 — 系统环境

```bash
sudo apt-get update && sudo apt-get upgrade -y
sudo apt-get install -y build-essential python3-dev python3-pip python3-venv \
    git nginx certbot python3-certbot-nginx curl wget

# TA-Lib C 库
wget http://prdownloads.sourceforge.net/ta-lib/ta-lib-0.4.0-src.tar.gz
tar -xzf ta-lib-0.4.0-src.tar.gz
cd ta-lib && ./configure --prefix=/usr && make -j$(nproc) && sudo make install
cd .. && rm -rf ta-lib ta-lib-0.4.0-src.tar.gz && sudo ldconfig

# Node.js 20 LTS
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Step 2 — 拉取代码 & 安装依赖

```bash
cd /opt
git clone https://github.com/ls569333469/BTC-PM.git btc-pm && cd btc-pm

# 后端
cd backend && python3 -m venv .venv && source .venv/bin/activate
pip install --upgrade pip
pip install fastapi uvicorn httpx apscheduler numpy pandas TA-Lib aiohttp

# 前端
cd ../frontend && npm install && npm run build
```

### Step 3 — Systemd 服务

```bash
sudo tee /etc/systemd/system/btc-backend.service << 'EOF'
[Unit]
Description=BTC Chanlun Analyzer Backend
After=network.target
[Service]
Type=simple
WorkingDirectory=/opt/btc-pm/backend
Environment="PATH=/opt/btc-pm/backend/.venv/bin:/usr/bin"
ExecStart=/opt/btc-pm/backend/.venv/bin/python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
Restart=always
RestartSec=3
[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload && sudo systemctl enable btc-backend && sudo systemctl start btc-backend
```

### Step 4 — Nginx

```bash
sudo tee /etc/nginx/sites-available/btc-pm << 'EOF'
server {
    listen 80;
    server_name _;
    root /opt/btc-pm/frontend/dist;
    index index.html;

    location / { try_files $uri $uri/ /index.html; }
    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
    location /ws/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_read_timeout 86400;
    }
}
EOF

sudo ln -sf /etc/nginx/sites-available/btc-pm /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default
sudo nginx -t && sudo systemctl reload nginx
```

### Step 5 — HTTPS（可选，需域名）

```bash
sudo certbot --nginx -d btc.yourdomain.com
```
