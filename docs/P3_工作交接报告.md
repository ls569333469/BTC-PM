## ⚠️ 已知限制

1. **综合评分相同**: 所有时间框架共享同一个 compositeWinRate (评分基于整体市场分析一次性计算)
2. **Polymarket slug查询**: 大概率失败 → 自动使用 fallback 纯缠论指南
3. **首次加载**: 投注指南部分约需 ~40-50 秒
4. **composite_correct**: 回测中全部为 null，因尚无足够的综合操作数据
